Remove links in comments based on article publication date and comment length.

== Description ==

This plugin goal is to prevent manual spammy links in comments. 

1. It automatically remove links into comments, on articles posted more than than three months ago.
2. If someone want to post links in a comment, he must write fifty words to write one link, in the limitation of three links. Beyond, links will be automatically removed (and switched by the anchor links, without url scheme)

These moderation rules can be overrided in the Settings->Discussion screen of the admin dashboard.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/advanced-remove-links-in-comments` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Discussion screen to configure the plugin
